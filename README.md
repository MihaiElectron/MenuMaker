# Projet 11 – Concevez une application de menu pour restaurateurs

## Contexte

Ce projet a été réalisé dans le cadre du parcours **Intégrateur Web OpenClassrooms**.

L'objectif est de préparer le développement de **Menu Maker**, une application destinée aux restaurateurs leur permettant de créer, gérer et exporter leurs menus numériques.

Le projet ne comporte pas de développement applicatif. Il consiste à produire l'ensemble des livrables de préparation nécessaires à la mise en œuvre du projet : spécifications techniques, organisation des tâches, veille technologique et présentation de la solution.

---

## Objectifs du projet

* Rédiger les spécifications techniques d'une application web.
* Découper les fonctionnalités en tâches exploitables par une équipe de développement.
* Mettre en place une veille technologique.
* Suivre l'avancement du projet à l'aide d'un outil de gestion Agile.
* Présenter les choix techniques retenus.

## Architecture technique proposée

### Front-end

* React
* React Router
* HTML5 / CSS3
* Accessibilité WCAG

### Back-end as a Service

* Supabase Auth
* Supabase Database (PostgreSQL)
* Supabase Storage
* API REST générée automatiquement

### Hébergement

* Front-end : Vercel ou Netlify
* Services back-end : Supabase
